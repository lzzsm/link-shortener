# Encurtador de Links

Um aplicativo web moderno para criar, gerenciar e acompanhar links encurtados com análise detalhada de acessos.

## 🚀 Tecnologias

- **React** - Biblioteca para construção da interface
- **Vite** - Build tool rápido e moderno
- **Firebase** - Backend, autenticação e banco de dados em tempo real
- **Analytics** - Rastreamento e análise de acessos aos links

## ✨ Recursos

- ✂️ Encurtador de URLs personalizado
- 📊 Dashboard com estatísticas de cliques
- 🔍 Análise detalhada de acessos (geo-localização, dispositivos, horários)
- 🔐 Autenticação segura com Firebase
- 📱 Interface responsiva
- 🎨 Design moderno e intuitivo

## 📦 Instalação

```bash
npm install
```

## 🏃 Executar Projeto

```bash
npm run dev
```

## 🔨 Build para Produção

```bash
npm run build
```

## 📝 Configuração Firebase

Configure suas credenciais do Firebase no arquivo de configuração do projeto para habilitar autenticação e armazenamento de dados.
